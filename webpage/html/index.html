Welcome to this website on art through continued functions.
By applying functions over and over again to a subset of the complex plane one can generate images by displaying the angle or radius of the limiting value (given that this exists). The resulting images can exhibit everything from rather simple, two color shapes to highly complex, fractal-like structures. Since the base function that generates the images is highly customizable, there is a nearly endless room of possibilities. It feels almost like exploring unknown continents, never previously seen by man!
This website provides

  • A sample gallery of pictures and gifs
  • A section on the theory and numerics behind the calculations
  • All the tools you need to start playing around and creating your own art
Have fun with CFA!
